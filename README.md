# 📊 Retail Sales Performance & Profitability Analysis

## 📌 Overview
This project analyzes retail sales data using SQL, Python, and Power BI to uncover business insights.

## 🛠️ Tools Used
- SQL (MySQL)
- Python (Pandas, Matplotlib)
- Power BI

## 📊 Key Insights
- High discounts negatively impact profit
- Sales peak during certain months (seasonality)
- Some products generate losses despite high sales
- Revenue is concentrated among top customers
